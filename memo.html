Mémo Vocabulaire

HTML Vocabulaire
  • Element : Définit un objet en HTML
    • Exemple: paragraphe(p), titre(h1, h2,...), hyperlien (a), mais aussi html, body, div, span,...

  • Balise : Un élément est représenté dans une fichier HTML par une balise
    • Tag en anglais
    • Toujours en minuscule
    • balise ouvrante et balise fermante
    • Il existe des élément qui n'ont pas besoin de balise fermante (br, img,...), ce sont des balises orphelines
  • Contenu : Le contenu d'un élément HTML
    • Situé entre la balise ouvrante et la fermante
    • Dans un fichier HTML on ne doit pas trouver du contenu non-balisé
    • Le contenu est la partie qui sera visible par l'utilisateur

  • Attribut : Permet d'ajouter des paramètres à nos éléments

    • Toujours en minuscule
    • Un élément peut avoir plusieurs attributs
    • Un attribut va souvent avoir une valeur

CSS Vocabulaire
  • Règle de style : Ensemble des règles de style appliqué sur un sélecteur
    • Voir exemple
    • Une règle de style, peut contenir plusieurs déclarations (voir img)

  • Sélecteurs : Détermine le(s) élement(s) sur le(s)quel(s) on applique les styles
    • Un sélecteur peut prendre plusieurs formes :
    • Nom de balise : h1, p, body,...
    • Nom d'une classe : .maClasse (utilisation du point)
    • Nom d'un id : #monId (utilisation du #)
    • Sélecteurs avancés : Liste W3C, Mémo

  • Propriété : Utilisé pour modifier le style d'un aspect particulier d'un élément
    • Il existe un tas de propriétés CSS : color, font-size, width, display,...
    • Une propriété CSS aura toujours une valeur: une couleur, une taille, un mot clé,...
